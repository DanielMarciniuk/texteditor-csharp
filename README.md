# TextEditor (C# Console)

## Português

Aplicação simples de **editor de texto em console** desenvolvida em **C#**.  
Permite criar, abrir, salvar e deletar arquivos diretamente pelo terminal.

### Funcionalidades

- Criar novo arquivo de texto
- Abrir e visualizar arquivos existentes
- Salvar texto em um arquivo
- Deletar arquivos
- Interface simples baseada em menu

### Tecnologias

- C#
- .NET
- Console Application
- Manipulação de arquivos com `System.IO`

### Como usar

1. Execute o programa.
2. Escolha uma opção no menu:
    - **1** Abrir arquivo
    - **2** Criar novo arquivo
    - **3** Deletar arquivo
    - **0** Sair
3. Informe o caminho do arquivo quando solicitado.

## English

Simple **console text editor** built with **C#**.  
Allows creating, opening, saving, and deleting text files directly from the terminal.

### Features

- Create new text files
- Open and read existing files
- Save text to a file
- Delete files
- Simple menu-based interface

### Technologies

- C#
- .NET
- Console Application
- File handling using `System.IO`

### How to use

1. Run the program.
2. Choose an option from the menu:
    - **1** Open file
    - **2** Create new file
    - **3** Delete file
    - **0** Exit
3. Enter the file path when requested.
